This example demonstrates the following aspects of the web component:

  • It behaves as expected in a flex layout. Play with the size of your viewport to confirm.
  • CSS containment still has to be enabled in order not to get rogue scrollbars, especially with a small viewport.

NOTE: on Safari, toggling the checkbox has no effect until the viewport gets resized or the component's CSS transform* is touched.

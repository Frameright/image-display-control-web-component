This example demonstrates the following aspects of the web component:

  • Zooming in on a region by using CSS transform* and clip-path would normally cause an overflow and thus unwanted scrollbars. To prevent this, the web component sets CSS containment (contain: paint;) on its parent. This does not prevent scrollbars altogether: the scrollbars caused by other elements are still there.
  • This behavior can be disabled by setting the data-disabled="css-contain" HTML attribute. Test it by clicking the Enable CSS containment checkbox.
